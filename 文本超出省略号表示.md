## 文本超出部分省略

首先要有宽度

三个属性：

1. overflow: hidden; //超出范围隐藏
2. white-space: nowrap; //超出部分不换行，撑开
3. text-overflow: ellipsis; //超出部分显示三个点

```
.tool-name {
            overflow: hidden;
            white-space: nowrap;
            text-overflow: ellipsis;
            word-break: keep-all;
        }
```
